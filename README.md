Proyek ini merupakan hasil dari Percobaan 2 pada Praktikum Jaringan Komputer, dengan judul Build a Switch and Router Network.
Tujuan dari percobaan ini adalah untuk membangun dan mengonfigurasi jaringan dasar yang terdiri dari satu router, satu switch, dan dua PC agar dapat saling terhubung menggunakan alamat IPv4 dan IPv6.

âš™ï¸ Konfigurasi yang Dilakukan
Dalam percobaan ini, dilakukan beberapa tahap konfigurasi, yaitu:
Menentukan topologi jaringan di Cisco Packet Tracer.
Memberikan alamat IP untuk setiap PC dan interface router.
Mengonfigurasi router (R1) meliputi hostname, password, interface, IPv6, banner, dan penyimpanan konfigurasi.
Mengonfigurasi switch (S1) dengan hostname, VLAN 1, dan default gateway.
Melakukan pengujian konektivitas (ping) untuk memastikan komunikasi berhasil.
